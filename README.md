# Time

# Timer Pro - Professional Timer Management System

## Быстрая установка

**Просто запусти `setup.bat` и всё!**

Скрипт автоматически:
- ✅ Проверит наличие Node.js
- ✅ Установит Node.js если его нет
- ✅ Установит все зависимости
- ✅ Соберёт portable EXE
- ✅ Скопирует TimerPro.exe на рабочий стол

## Использование

После установки:
1. Запусти **TimerPro.exe** с рабочего стола
2. Приложение можно перемещать куда угодно - оно полностью portable
3. Все данные хранятся рядом с exe файлом

## Функции

- ✅ Множественные таймеры
- ✅ Имена таймеров 
- ✅ Системные уведомления
- ✅ Звуковые сигналы
- ✅ System tray (сворачивается в трей)
- ✅ Автообновления
- ✅ Строгий профессиональный дизайн

## Дизайн

- Тёмная тема (#0a0a0a)
- Золотой акцент (#c8a050)
- Моноширинный шрифт
- Плавные анимации
- Кастомный titlebar

## Структура проекта

```
electron-app/
├── main.js              # Главный процесс
├── preload.js           # IPC мост
├── package.json         # Конфигурация
├── setup.bat            # Автоматическая установка
├── renderer/
│   ├── index.html       # UI
│   ├── styles.css       # Стили
│   └── app.js           # Клиентская логика
└── assets/              # Иконки
```

## Требования

- Windows 7/8/9/10/11
- Интернет для первой установки (если нет Node.js)
- ~150 MB свободного места

## Примечания

- При первом запуске setup.bat может занять 2-3 минуты
- Node.js устанавливается только если его нет
- После сборки папку electron-app можно удалить
- TimerPro.exe полностью автономен
